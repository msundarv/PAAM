# PAAM - Personal AI Asset Manager
A multi-agent system that tracks and consolidates your investments, provides real-time market insights, and keeps you informed - enabling seamless, efficient portfolio management.

## Prerequisites

- **Python 3.14** or higher
- Required dependencies listed in [requirements.txt](https://github.com/Msundarv/PAAM/blob/main/requirements.txt)

Install dependencies:
```bash
pip install -r requirements.txt
```

## Config

1. Create a `.env` file in the project root directory by copying `.env.example` and adding your required API keys and credentials.
2. Define your assets in `/data/assets.yml` following the schema provided in the example file.

## Quick Start

To run the application locally:

```bash
python -m streamlit run paam.py
```
